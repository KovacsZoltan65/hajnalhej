# Áttekintés

## Mi a Hajnalhéj rendszer célja?

A Hajnalhéj Bakery rendszer egy pékségi rendelési és adminisztrációs platform. Segít abban, hogy a vásárlók könnyen rendeljenek, a pékség pedig áttekinthetően kezelje a termékeket, rendeléseket, alapanyagokat, készletet, beszerzést és riportokat.

## Kinek készült?

- Vásárlóknak: heti menü böngészés, kosár, checkout, fiók.
- Adminoknak: teljes admin felület kezelése.
- Operátoroknak: rendelések, készlet, beszerzés, leltár napi kezelése.
- Vezetőknek: profit, conversion, CEO és security dashboardok.

## Fő modulok röviden

- Termékek: pékségi termékek törzsadatai.
- Kategóriák: termékek csoportosítása.
- Heti menü: az adott időszakban rendelhető kínálat.
- Rendelések: vásárlói rendelések feldolgozása.
- Alapanyagok: készlet, minimum készlet, becsült költség.
- Receptek / BOM: termékek alapanyagigénye.
- Beszerzések: beszállítói vásárlások és készletbevétel.
- Inventory: készletmozgások, selejt, korrekció.
- Leltár: tényleges készlet ellenőrzése.
- Procurement Intelligence: utánrendelési és ártrend döntéstámogatás.
- Dashboardok: vezetői, profit, conversion és security nézetek.

## Napi tipikus használat

1. Rendelések áttekintése.
2. Rendelések státuszának frissítése.
3. Készlet dashboard ellenőrzése.
4. Beszerzési intelligencia figyelmeztetések átnézése.
5. Szükség esetén beszerzési draft generálása.
6. Draft beszerzés ellenőrzése, szerkesztése.
7. Beérkezéskor beszerzés könyvelése.
8. Selejt vagy készletkorrekció rögzítése.
9. Profit és vezetői dashboardok ellenőrzése.

## Fontos alapelv

A rendszerben a **Minimum készlet** jelenti az utánrendelési szintet. Ha egy alapanyag készlete ez alá csökken, a rendszer beszerzési figyelmet javasol.
